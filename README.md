Στόχος της εργασίας, είναι να μοντελοποιηθεί η εξέλιξη πάνω σε ένα
κυψελοειδές αυτόματο, η συμπεριφορά των κυψελών βασίζεται στο κλασσικό
παιχνίδι «Πέτρα-Ψαλίδι-Χαρτί». Οι δύο πληθυσμοί στο αυτόματό κυψελοειδές,
εκφράζουν συμπεριφορά που εξαρτάται από την δράση παικτών, με την ενσωμάτωση
του αλγορίθμου ανταμοιβής-ποινής για την μάθηση του παιχνιδιού εναντίον των
γειτόνων τους. Επιπλέον, το αυτόματο και τα μέλη του πληθυσμού των παικτών του
έχουν περιορισμούς με αποτέλεσμα η μόνη μορφή μάθησης του αντιπάλου να
βασίζεται στην μνήμη της τελευταίας στρατηγικής που έχουν οι συγκεκριμένοι
παίκτες. Επίσης, ο μηχανισμός μνήμης είναι αυτός που επιτρέπει στους
συμμετέχοντες των πληθυσμών να διατηρούν μια εγγραφή της τελευταίας πράξης του
αντιπάλου τους. Χωρίζονται δύο ομάδες, οι οποίες προχωρούν σε ένα κυψελοειδές
αυτόματο, με τους παίκτες τους να εφαρμόζουν την ίδια στρατηγική για την
εκμάθηση του παιχνιδιού με μία ουσιώδη διαφοροποίηση. Πιο συγκεκριμένα, η
Ομάδα 1 έχει μια μεγαλύτερη ταχύτητα μάθησης από την Ομάδα 2, προκειμένου να
γίνουν αντιληπτές οι επιπτώσεις πάνω στην εξέλιξη του συγκεκριμένου κυψελοειδούς
αυτόματου.
